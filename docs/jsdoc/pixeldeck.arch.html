<static> getResPathFor( aPath, replacementExt )

Description

Convert any path into a resource path, optionally changing the extension This will search the following paths:

  1. None: just try not modifying aPath's directory
  2. "configured" resource directory.
  3. "current working directory" as reported by node.
Parameters
Name Type Description
aPath string
replacementExt string

(optional), giving '' strips off any extension
Details

"current working directory" may need to be scrapped so that client/server side works the same.
